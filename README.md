The neural architecture used in the practical part of this thesis is the BEATs transformer, which has its own dedicated GitHub repository.
Accordingly, the complete BEATs source code can be found in the files contained in this repository.

This experiment consisted in removing the projection layer of the BEATs classifier (which operates within the AudioSet ontology) in order to perform classification within a different ontology: UCS.

UCS is a sound classification method widely used in audiovisual post-production.
This thesis represents an attempt to automate this classification process, which is a time-consuming task for audio post-production professionals.

The present work should be considered a proof of concept, as the experiment was carried out and demonstrated to be feasible on 6 of the 752 UCS classes.
